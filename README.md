# shwty
Простой сервис для сокращения ссылок, написанный на Node.js + Express.
## Технологии
- Node.js
- Express
- JavaScript (ES6)
- dotenv
- helmet + cors
- Tailwind
- GSAP

## Как запустить локально

   Склонируй репозиторий:
   ```powershell
   git clone https://github.com/Naknad/shwty-URL_Shortener.git
   cd shwty-URL_Shortener
   npm install
   npm run dev

   открываем в браузере: http://localhost:3000
